#### 基础知识

redis默认有16个数据库,默认使用的是第0个，可用select n来切换数据库

redis.conf
```
databases 16
```

查看数据库key的数量
```
dbsize
keys *
```

清空当前数据库
```
flushdb
```

清空全部数据库
```
flushall
```

**redis是单线程的，基于内存操作，CPU不是性能瓶颈，瓶颈是机器的内存和带宽，既然可以使用单线程来实现，就使用单线程了。**

多线程要有CPU上下文切换，不一定比单线程效率高，对内存系统来说，如果没有上下文切换效率就是最高的，多次读写都是在一个cpu上进行的。
